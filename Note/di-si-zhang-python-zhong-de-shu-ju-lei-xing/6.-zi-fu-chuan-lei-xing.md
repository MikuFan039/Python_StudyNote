# 6.字符串类型

### 字符串类型（ `str` ）

**连续的字符序列**，为不可变数据类型，可以**表示计算机所能识别的一切字符**

字符串的界定符：**单行**可使用**单引号或双引号**，**多行**应使用**三引号**

### 示例代码

字符串类型的使用

{% code overflow="wrap" lineNumbers="true" %}
```python
city='天津'
address="天津市宝坻区香江大街3号"
print(city)
print(address)
# 多行字符串
info='''地址:天津市宝坻区香江大街3号
    收件人:杨淑娟
    手机号:18600000000
'''
info2="""地址:天津市宝坻区香江大街3号
    收件人:杨淑娟
    手机号:18600000000

"""
print(info)
print('------------------')
print(info2)
```
